1. С помощью цикла while вывести все простые числа в промежутке от 0 до 100

    Решение:

    Простое число — натуральное (целое положительное) число, имеющее ровно два различных натуральных делителя — единицу и самого себя

    
    let prime = 1;
    let primeArr = [];

    while (prime <= 100) {
        if (prime == 1) {
            primeArr.push(1)
        } else {
            let result = 2;
            let i = 2;
            while (prime % i !== 0 ){
                i += 1;
                result +=1;
                } 
            if (result == prime) {
                primeArr.push(result);
            }
        }
        prime += 1;
    }

    console.log(primeArr);
  2. С помощью цикла do…while написать функцию для вывода чисел от 0 до 10, чтобы результат выглядел так: 0 – это ноль 1 – нечетное число 2 – четное число 3 – нечетное число … 10 – четное число

    Решение: 

        let a = -1;
    do {
        a += 1;
        if (a === 0) {
            console.log(a, " - это ноль");
        } else if (a % 2 ===0) {
            console.log(a, " - это четное число");
        } else {
            console.log(a, " - это нечетное число");
        }
    } while (a < 10);
